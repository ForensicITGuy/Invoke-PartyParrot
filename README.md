# Invoke-PartyParrot
Bringing the Party Parrot to PowerShell!

## Usage

Download the script by cloning this repo or or obtaining the raw text.

    ./Invoke-PartyParrot.ps1

When it's time to go back to work, press 'q' to end the party.
